In 2019 I started building this website,
learning HTML and CSS along tha way as a self-learner.
The website has had multiple codenames [the origin of which is a secret], each for every major version: - SprattoZero - First instance of this website, run at home on a Raspberry Pi Zero W,
which I manually configured to host an Apache2 website. This was only accessible on my local network
- SprattoUno - This version was hosted on a VPS which I bought in mid 2019 for a one year subscription.
This was the first version of the website to be accessible from everywhere thanks to a .tk domain. The VPS was otherwise configured just as the original SprattoZero on Raspberry.
- SprattoUno+ - This version was manually configured with a .com domain and a full
cloudflare-backed security with https. Same setup as before for the rest.
- SprattoDue - This version came to life in mid 2020 after the VPS subscription expired.
Same website, same domain but hosted on Github and with its own https.
